# 🤖 Бот-помощник для Битрикс24

Бот для автоматических ответов в личных сообщениях Битрикс24. Помогает отвечать на вопросы по статусам объектов закупок, фильтрует нерелевантные запросы и уведомляет ответственного сотрудника при сложных вопросах.

## 📋 Возможности

| Функция | Описание |
|---------|----------|
| ✅ **Автоответ на приветствия** | Ответы на "привет", "спасибо", "пока" |
| ✅ **Статусы по объектам** | Запрос информации по объекту (например, "статус по объекту Павлово") |
| ✅ **Диалоговый сценарий** | Бот сам спрашивает название объекта, если его не указали |
| ✅ **Словарь исключений** | Отклоняет нерелевантные вопросы (погода, анекдоты, цвет луны) |
| ✅ **Уведомления ответственного** | При сложных вопросах отправляет уведомление Дмитрию |
| ✅ **Память диалогов** | Запоминает, что пользователь уже спросил про объект |
| ✅ **Работа в личных сообщениях** | Не требует перехода в открытые линии |

## 🏗️ Технологии

- **Node.js** — среда выполнения
- **Vercel** — хостинг и серверless-функции
- **GitHub** — хранение кода и автоматический деплой
- **Bitrix24 REST API** — взаимодействие с порталом
